## Usage

- Export
  ```bash
  $ brew bundle dump
  ```
- Import
  ```bash
  $ brew bundle
  ```
- Check the list of packages in the Brewfile
  ```bash
  $ brew bundle list
  ```
- Uninstall all packages not listed in the Brewfile
  ```bash
  $ brew bundle cleanup
  ```
  <br>
